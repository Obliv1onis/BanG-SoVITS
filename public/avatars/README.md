# Character avatar sources

The character images in this directory were downloaded from the corresponding
character pages on the [BanG Dream! Wiki](https://bandori.fandom.com/wiki/Category:Characters)
on Fandom on 2026-07-17. The page mapping used for retrieval is recorded in
`scripts/fandom-avatar-pages.tsv`.

BanG Dream! and its characters belong to their respective rights holders. Wiki
page contributions are made available under Fandom's applicable licensing terms.
